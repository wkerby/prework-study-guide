Prework Study Guide

HTML

  • The head element contains info about the webpage
  • The body element represents the visible content shown to the user
  • Attributes are specified in elements, similar to parameters in Python methods or functions
  • Elements are slightly different than tags

CSS

  • Language used for styling a webpage

Git

  • Version control system
  • Used to document additions and changes to a code base
  • GitHub is a popular cloud-based git repository hosting service that renders collaboration amongst developers easier
  • git status returns branch in which we are currently working
  • git switch -c "branch name" creates a new branch and switches to it

JavaScript

  • Language used to develope logic and interactivity of a web application
